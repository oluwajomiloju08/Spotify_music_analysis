<img width="1579" height="105" alt="image" src="https://github.com/user-attachments/assets/643dc19e-4958-44f8-9d98-476a3579b018" /> ## Spotify Music Dataset Analysis
This project explores spotify streaming data to identify audience engagement trends, top-performing artists, and music popularity patterns using SQL, Excel and Power BI.
## Tools used
- SQL.
- Excel.
- Power BI.
 ## Objectives
 - Identify top-performing artists and tracks.
- Analyze genre trends.
- Monitor audience engagement metrics.
 - Visualize streaming performance.
 ## Dashboard Features
 - Popularity scores tracking.
 - Top artists analysis.
 - Genre performance.
 - Audience engagement KPIs.
 ## Key Insights
 - Playlists appearances positively impacted streaming performances.
- Certain genre consistently outperormed others.
- Some less popular Artists still have standout songs.
 ## Recommendations
 - Build playlists around strong genres seen in the data.
 - Promote rising artists who have good songs but low popularity.
 - Use release-year patterns to understand how music trend evolve.
